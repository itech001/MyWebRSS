Presentation

MyWebRSS is a web RSS reader. It is OpenSource and can be installed on every server with PHP and MySQL.

The client side is HTML5 + JavaScript, for the best compatibility on mobiles platforms (iOS, Android, FirefoxOS) and desktop computers.

The client side gets all informations from an API, developped in PHP and that uses a MySQL database. The datas are send in JSON.

Versions

A stable version of the client is available at http://html5.mywebrss.net from a web-browser on Desktop (except Internet Explorer) or on a Smartphone (Android / iOS / etc.). The web-browser needs HTML5 + JavaScript support.

A stable version of the API is available at https://api.mywebrss.net and uses the same database as the official client. All connections are made through HTTPS, with a signed certificate.

If you want to install it on your server, you can get the source code from Github: https://github.com/ybulach/MyWebRSS.

Privacy

To use this application, an account is needed, identified by an email address. The authentication is made through Persona (https://login.persona.org), thus this application doesn't have any access to your password.

All data exchanges between the application and the official stable server (from http://html5.mywebrss.net, located in France) use HTTPS with a signed certificate from StartSSL.

Report Bugs

As the source code is available on Github, if you find bugs in the application, you can open issues at https://github.com/ybulach/MyWebRSS.
